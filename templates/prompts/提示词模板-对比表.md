# 对比表/信息图表示提示词模板

**类型：** 对比表/信息图表
**用途：** 文内配图，展示对比关系
**尺寸：** 1024x1024（正方形）

---

## 模板结构

```
现代极简主义+科技信息图表风格。
[对比表画面描述]
清晰的对比关系，直观的视觉表达，专业的信息图表质感。
```

---

## 示例 1：技术主权 vs 依赖外部

```
现代极简主义+科技信息图表风格。
左右分栏的对比表格：左侧是「技术主权」，包含自主可控、数据安全、灵活定制三个元素；右侧是「依赖外部」，包含受制于人、数据风险、成本高昂三个元素。每个元素都有简洁的图标，整体清晰直观。
清晰的对比关系，直观的视觉表达，专业的信息图表质感。
```

---

## 示例 2：不同模型对比

```
现代极简主义+科技信息图表风格。
三列对比表格：分别是「模型A」「模型B」「模型C」，每列包含准确性、速度、成本三个维度的对比，用不同颜色的条形图展示，整体清晰直观。
清晰的对比关系，直观的视觉表达，专业的信息图表质感。
```

---

## 注意事项

1. **只保留画面内容描述和基本风格**
2. **不要添加颜色代码、像素高度等技术约束**
3. **不要使用"标题"这个词（改成"文字"）**
4. **不要过度强调留白位置**
