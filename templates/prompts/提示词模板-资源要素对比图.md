# 资源要素对比图提示词模板

**类型：** 资源要素对比图
**用途：** 文内配图，展示资源要素对比
**尺寸：** 1024x1024（正方形）

---

## 模板结构

```
现代极简主义+科技信息图表风格。
[资源要素对比画面描述]
清晰的资源要素对比，直观的视觉表达，专业的信息图表质感。
```

---

## 示例 1：时间 vs 成本 vs 质量

```
现代极简主义+科技信息图表风格。
三角形的三个顶点分别是「时间」「成本」「质量」，三角形内部是一个三维坐标系，展示三者之间的平衡关系，每个顶点都有简洁的图标，整体清晰直观。
清晰的资源要素对比，直观的视觉表达，专业的信息图表质感。
```

## 示例 2：人力 vs 技术 vs 资金

```
现代极简主义+科技信息图表风格。
三个圆圈分别代表「人力」「技术」「资金」，三个圆圈有部分重叠，重叠区域显示协同效应，每个圆圈都有简洁的图标，整体清晰直观。
清晰的资源要素对比，直观的视觉表达，专业的信息图表质感。
```

## 注意事项
