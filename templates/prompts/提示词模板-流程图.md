# 流程图/时间线提示词模板

**类型：** 流程图/时间线
**用途：** 文内配图，展示流程或时间关系
**尺寸：** 1024x1024（正方形）

---

## 模板结构

```
现代极简主义+科技信息图表风格。
[流程图/时间线画面描述]
清晰的流程关系，直观的时间表达，专业的信息图表质感。
```

---

## 示例 1：内容生产工作流

```
现代极简主义+科技信息图表风格。

清晰的流程关系，直观的时间表达，专业的信息图表质感。
```
