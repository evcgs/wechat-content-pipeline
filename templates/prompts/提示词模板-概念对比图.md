# 概念对比图提示词模板

**类型：** 概念对比图
**用途：** 文内配图，展示概念对比
**尺寸：** 1024x1024（正方形）

---

## 模板结构

```
现代极简主义+科技信息图表风格。
[概念对比画面描述]
清晰的概念对比，直观的视觉表达，专业的信息图表质感。
```

---

## 示例 1：技术主权 vs 依赖外部

```
现代极简主义+科技信息图表风格。
画面中央是一个天平，左侧是「技术主权」（包含自主可控、数据安全、灵活定制），右侧是「依赖外部」（包含受制于人、数据风险、成本高昂），天平倾向技术主权一侧，整体传达出技术自主的重要性。
清晰的概念对比，直观的视觉表达，专业的信息图表质感。
```

---

## 示例 2：本地化 vs 云端

```
现代极简主义+科技信息图表风格。
左右两个圆圈：左侧是「本地化」（包含数据安全、离线可用、自主可控），右侧是「云端」（包含成本低、易扩展、按需付费），两个圆圈有部分重叠，整体传达出各有优势的概念。
清晰的概念对比，直观的视觉表达，专业的信息图表质感。
```

---

## 注意事项

1. **只保留画面内容描述和基本风格**
2. **不要添加颜色代码、像素高度等技术约束**
3. **不要使用"标题"这个词（改成"文字"）**
4. **不要过度强调留白位置**
