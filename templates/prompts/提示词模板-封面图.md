# 封面图提示词模板

**类型：** 封面图
**用途：** 文章封面图，吸引点击
**尺寸：** 2560x1440（横幅，16:9）

---

## 模板结构

```
现代极简主义+科技信息图表风格。
[画面描述]
干净的背景，柔和的渐变，清晰的视觉层次，杂志封面的质感，专业而有吸引力。
```

---

## 示例 1：医学大模型「不可能三角」

```
现代极简主义+科技信息图表风格。
画面中央是一个发光的三角形，三个顶点分别是「可及性」「质量」「成本」三个元素，三角形中心闪烁着AI神经网络的光芒，背景是纯净的渐变蓝色，整体传达出技术平衡的概念。
干净的背景，柔和的渐变，清晰的视觉层次，杂志封面的质感，专业而有吸引力。
```

## 注意事项

1. **只保留画面内容描述和基本风格**
2. **不要添加颜色代码、像素高度等技术约束**
3. **不要使用"标题"这个词（改成"文字"）**
4. **不要过度强调留白位置**
